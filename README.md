**RussianTimeExpansion** — это высокопроизводительное дополнение для **PlaceholderAPI**, созданное для красивого оформления текста на серверах Minecraft. 

Изначально плагин создавался для локализации дат (дни недели и месяцы на русском), но теперь это мощный инструмент преобразования любого текста и сторонних плейсхолдеров в стильный кириллический **Small Caps** (ᴍᴀлᴇʜьᴋиᴇ ɜᴀᴦлᴀʙʜыᴇ бyᴋʙы). Идеально подходит для кастомизации Scoreboard, TAB и GUI меню без использования ресурспаков.

## 🌟 Особенности (Features)
* **Small Caps маппинг:** Превращает любой текст или вложенный плейсхолдер в красивый шрифт.
* **Русская локализация:** Автоматический перевод дней недели и месяцев на русский язык.
* **Конфигурация TimeZone:** Позволяет задать жесткий часовой пояс (например, `Europe/Moscow`), что решает проблему с отображением неправильного времени, если сервер физически хостится в другой стране.
* **TPS Friendly & Zero-Allocation:** Замена символов происходит через прямой доступ к массиву (`char[]`), а даты кэшируются через потокобезопасный `DateTimeFormatter`. Плагин не нагружает Garbage Collector и работает за наносекунды.

## 📋 Плейсхолдеры (Usage)

### 1. Форматирование текста (Small Caps)
Вы можете оборачивать **любые** сторонние плейсхолдеры. Важно: внутренние плейсхолдеры должны использовать фигурные скобки `{}` вместо `%`.

| Плейсхолдер | Описание | Пример вывода |
| :--- | :--- | :--- |
| `%rutime_caps_<текст>%` | Переводит обычный текст в Small Caps | `пᴘиʙᴇᴛ, ᴄᴇᴘʙᴇᴘ!` |
| `%rutime_caps_{player_name}%` | Переводит ник игрока | `sᴛᴇvᴇ` |
| `%rutime_caps_{vault_eco_balance} монет%` | Комбинация плейсхолдера и текста | `1000 ᴍᴏʜᴇᴛ` |

### 2. Время и даты
Формат времени задается стандартными паттернами Java (`SimpleDateFormat` / `DateTimeFormatter`).

| Плейсхолдер | Описание | Пример вывода |
| :--- | :--- | :--- |
| `%rutime_now%` | Быстрое время в формате по умолчанию (из конфига) | `19.02.2024 15:30` |
| `%rutime_format_HH:mm%` | Только часы и минуты | `15:30` |
| `%rutime_format_dd MMMM%` | День и месяц на русском | `19 ȹᴇʙᴘᴀля` |
| `%rutime_format_EEEE%` | День недели на русском | `пᴏʜᴇдᴇльʜиᴋ` |

## ⚙️ Установка и Настройка (Installation)

1. Скачайте последний релиз (`.jar` файл).
2. Поместите файл в папку `/plugins/PlaceholderAPI/expansions/`.
3. Введите команду `/papi reload` в консоли или игре.
4. В файле `/plugins/PlaceholderAPI/config.yml` автоматически появится секция `rutime`.

**Пример генерируемого конфига:**
```yaml
expansions:
  rutime:
    timezone: Europe/Moscow
    default_format: dd.MM.yyyy HH:mm
